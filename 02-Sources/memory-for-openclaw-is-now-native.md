# Memory for OpenClaw is now Native!

## Source
[Original Link](https://x.com/kevinnguyendn/status/2037730451255959840?s=46)

## Summary
OpenClaw has released a native memory plugin that integrates directly into agent workflows, moving beyond skill-based memory. It features a three-layer memory architecture with Git-like organization and achieves 92.2% retrieval accuracy while being local-first and portable. This matters for building multi-agent systems as it enables persistent memory for complex workflows like trading and prediction market analysis.
